## Key Assessment Areas

### Dark Mode
- Text contrast (amber-600 readable on dark background?)
- Form input borders visible?
- Card backgrounds distinct?
- Status badges readable?

### Tier Gating
- SIMPLE: blocked from analytics, batch-ops, command-center, brand-kit, workspace
- PRO: blocked from workspace only (shows TEAMS upsell)
- TEAMS: full access
- Upsells present and on-brand?

### Accessibility
- TAB navigates all elements?
- Focus states visible?
- Icon-only buttons have aria-labels?
- Mobile nav icons labeled?

### General
- All pages load (✅/⚠️/❌)
- No broken images
- Nav item count reasonable
- No console errors
- Badge explosion on sale cards?
