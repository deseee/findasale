### E6. Autonomous Instruction Pattern (Steelman Method) — P2 [WORKFLOW]

From forum: A user's workflow for getting complex projects done in one shot:
1. Brainstorm with Claude first (problem space, not solution)
2. Have Claude write the build plan
3. Iterate on the plan together
4. Convert plan into autonomous execution instructions ("I'm not doing anything — you figure it all out")
5. Have Claude review its own instructions (surfaces 10–15 issues)
6. Steelman: Claude argues against its own criticism (kills ~50% of false alarms)
7. Apply surviving fixes, fresh chat, repeat steelman once more
8. Execute — reported 30-minute uninterrupted runs that one-shot complex projects

Also: "Have Claude Opus 4.6 create a plan for each agent. Specify the model for each agent — for some, Sonnet or Haiku is more than enough."

**Owner:** `cowork-power-user` to evaluate and prototype, `findasale-workflow` to integrate
