### Topic 3: UX Patterns — Broader Shopper Types Lens

**S235 Verdict:** Joybird patterns apply to estate sales (furniture focus). Room planner + condition tags.

**S236 Changes:**

| Shopper Type | Behavior | UX Needs | Priority |
|-------------|----------|----------|----------|
| **Estate Buyer** | Pre-planned, research-driven | Reputation badge, condition tags, 5 photos | P0 (Q2) |
| **Garage Shopper** | Spontaneous, deal-hunting, local | Map discovery, early-bird alerts | P1 (Q3) |
| **Auction Bidder** | Competitive, time-sensitive | Countdown timers, live updates, lot grouping | P2 (Q3) |
| **Consignment Buyer** | Quality-focused, repeat visitor | Personalization, brand filtering, seller profile | P3 (Post-Q3) |
| **Flea Market Browser** | Discovery-driven, social, in-person | Booth map, pre-event preview, local pickup | P4 (Post-2026) |

**Key Insight:** One-size-fits-all UX breaks for non-estate shoppers. Need sale-type-aware discovery:
- Estate sale filter selected → show reputation badges, confidence badges, shipping info
- Garage sale filter selected → show map, early-bird toggle, local pickup, deal indicators
- Auction filter selected → show countdown, current bid, lot grouping
- Consignment filter selected → show seller profile, personalization, brand filter
- Flea market filter selected → show booth location, pre-event inventory, vendor profile

**New Verdict Summary:**
- ✅ Condition Tags + Room Planner (Q2–Q3) — **BUILD NOW** (unchanged, applies to all categories)
- ✅ Sale-Type-Aware Discovery (Q3) — **BUILD NOW** (new: critical for garage/auction shoppers)
- ✅ 2D Room Planner (Q3) — **BUILD NOW** (unchanged)
- ⚠️ 3D AR Furniture (Q4) — **DEFER** (incremental to 2D)
- ⚠️ Seller Profiles + Social Features (Post-Q3) — **DEFER 2027** (Poshmark-style, for consignment focus)
