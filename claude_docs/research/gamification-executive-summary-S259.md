## THE CORE IDEA IN ONE SENTENCE

Shoppers climb permanent ranks (Initiate → Grandmaster) that unlock real perks (discounts, presales), while competing on seasonal leaderboards that reset every quarter without erasing their rank.

---

## WHY THIS WORKS BETTER THAN COMPETITORS

| **System** | **Permanent** | **Seasonal Reset** | **Functional Rewards** | **Ecosystem** |
|---|---|---|---|---|
| **Starbucks** | ✓ Stars | ✗ No | Cosmetic (badge) | Single-sided |
| **eBay** | ✓ Seller Tiers | ✗ No | Functional (fees) | Seller-only |
| **Poshmark** | ✓ Ambassador | ✗ No | Functional (revenue share) | Limited (top 2%) |
| **Etsy** | ✓ Seller Stars | ✗ No | Functional (algo boost) | Seller-only |
| **FindA.Sale** | ✓ Guild Ranks | ✓ Quarterly | Functional (discount + presales) | **Shopper ↔ Host** |

**Unique Advantage:** No competitor combines permanent rank + seasonal leaderboard. This solves the "veteran frustration cliff" that killed gamification in other platforms.

## THE SYSTEM AT A GLANCE

### Shopper Side: The 5 Ranks

| **Rank** | **Entry** | **Real Reward** | **Annual Cost/Value** |
|----------|-----------|---|---|
| **Initiate** | Day 1 | Baseline access | $0 |
| **Scout** | 500 XP (~$500 spent) | 5% Hunt Pass discount | $15/year |
| **Ranger** | 2,000 XP (~$2k spent) | 10% discount + 2h support SLA | $30/year + staffing |
| **Sage** | 5,000 XP (~$5k spent) | 15% discount + 24h presale access | $45/year + inventory scarcity |
| **Grandmaster** | 12,000 XP (~$12k spent) | 20% discount + free Hunt Pass annually | $60/year + staffing |

**XP comes from:**
- Purchases ($1 = 1 XP)
- Sale visits (5 XP each, max 2/day)
- Referrals (50 XP one-time)
- Collection specialties (25 XP per category)
- Seasonal challenges (50–500 XP)

**Key insight:** A typical active shopper reaches Scout in 6–12 months, Ranger in 2–3 years. Grandmaster is 4–5 year goal. This is **aspirational, not immediate.**

### Organizer Side: Host Ranks (Phase 2)

Parallel system for hosts (estate sale companies, yard sale operators, auction houses):

| **Host Rank** | **Entry** | **Fee Discount** | **Feature Access** |
|---|---|---|---|
| **Novice Host** | Day 1 | 0% | Standard tools |
| **Trusted Host** | 50+ sales, 4.5+ rating | 1% (9% instead of 10%) | Featured placement |
| **Elite Host** | 200+ sales, 4.8+ rating | 2% (8% instead of 10%) | Beta features |
| **Master Host** | 500+ sales, 4.9+ rating | 3% (7% instead of 10%) | Dedicated account manager |

**Interaction with Shoppers:**
- Sage/Grandmaster shoppers get 24–48h exclusive presale access to Master/Elite host sales
- Grandmaster shoppers can endorse host sales, boosting visibility
- Hosts can request top shoppers to "curate" specific collections (vintage, furniture, etc.)

### Seasonal Expeditions (4 Quarters)

Every quarter resets leaderboard but keeps permanent rank intact:

- **Q1 Spring Awakening** (March–May): Estate cleanings, seasonal renewal. "Spring Legend" badge for top 10.
- **Q2 Summer Adventure** (June–Aug): Travel season, regional hopping. "Summer Champion" title.
- **Q3 Fall Collection** (Sep–Nov): Specialization focus, holiday prep. "Fall Sage" featured in gift guides.
- **Q4 Winter Treasures** (Dec–Feb): Holiday gifts, year-end reflection. "Winter Legend" hall of fame.

Each season has 3–5 challenges. Top 100 finishers get permanent seasonal badges + leaderboard spot.

### Micro-Events (16–19 per year)

On top of seasonal expeditions, 1–3 week themed events tied to real-world calendar:

- **January:** New Year, New Collections + Vintage Rendezvous
- **February:** Valentine's Treasure Hunt + Black History Month Discoveries
- **March:** St. Patrick's Luck
- **April:** Spring Cleaning Sprint + Earth Day Treasures
- **May:** Graduation Gift Guide
- **June:** Pride Month Celebration
- **July:** Picnic Treasures + Vintage Fashion Fiesta
- **August:** Back-to-School Hunt
- **September:** Labor Day Deals
- **October:** Spooktacular Collectibles + Antique Appraisal Event
- **November:** Gratitude Gathering + Black Friday Flip
- **December:** Holiday Gift Quest + Year-End Reflection

Events offer bonus XP multipliers (1.5x–2x) and limited badges. They don't reset leaderboards; they add flavor to seasonal arcs.

## HOW IT DEFENDS AGAINST BOARD OBJECTIONS

### "This is too complicated."

**Response:** Casual players (60%) see one number: "Guild Points." Engaged players (30%) unlock the full rank ladder. Hardcore (10%) see leaderboards and challenges. Complexity is opt-in; simplicity is default. (See Section 6 of full spec for UX mockup details.)

### "It's just a badge system with better marketing."

**Response:** No other platform has this architecture. More importantly, rewards are functional (recurring discounts, operational cost, presale inventory access) not cosmetic. Rank discounts alone cost ~$45–60/year per Sage/Grandmaster. Not free.

### "Seasonal resets will frustrate long-term players."

**Response:** Permanent rank never resets. Your Sage status stays forever. Seasonal leaderboard is bonus engagement (optional to compete), not destructive reset. Analogous to Overwatch: seasonal rank resets, but account level persists.

## COMPETITIVE MOAT

**What we own that nobody else does:**

1. **Permanent + Seasonal Dual System:** eBay = permanent. Apex Legends = seasonal with permanent level. We have both, integrated. Defensible.
2. **Functional Rewards That Cost Money:** Presale access = scarcity. Priority support = staffing. Discounts = revenue. These aren't free badges.
3. **Ecosystem Coupling:** Shoppers unlock hosts' presales. Hosts can request top collectors. This is system-level, not slapped-on.
4. **Secondary Market Narrative:** "Collector's Guild" narrative works across estate sales, yard sales, auctions, flea markets, consignment. No competitor owns this positioning.

## TECHNICAL RISK (ARCHITECT SUMMARY)

| **Concern** | **Risk Level** | **Mitigation** |
|---|---|---|
| **Schema Complexity** | **LOW** | 5 new tables, straightforward relationships. No circular deps. |
| **XP Calculation Scale** | **MEDIUM** | Sync for purchases (critical path), async for visits/challenges. Redis cache for reads. |
| **Event System** | **LOW-MEDIUM** | Hardcoded registry (static) vs. dynamic DB (flexible). Start with static, migrate if needed. |
| **Rank Rewards Economics** | **LOW** | Fee discounts are built into Stripe. Presale = routing feature (15% effort increase). |
| **Data Integrity** | **MEDIUM** | Denormalization risk (totalGuildXP cache). Mitigation: DB triggers + nightly batch verify. |

**No hard blockers. All concerns are manageable.**
